02 / About

Code that ships.
Systems that scale.

I'm a Unity developer who came into games through a non-linear route — studying English Translation before teaching myself C# and Unity, then joining Maple Labs in 2022. Since then I've shipped 20+ mobile titles and helped build the internal tooling other devs on the team use every day.

My focus is on the layer between gameplay prototype and shipped product: reusable systems, editor tools for designers, integrating monetization and analytics (AppLovin, Firebase, Adjust, Tenjin), and mentoring newer teammates through workshops and code review. I care about code that other people can read, reuse and extend — not just code that works once.

Outside the editor I speak English, Chinese and Cantonese, and I'm always running self-directed research on new Unity features — DOTS, UI Toolkit, shader work — so the next title ships a little faster than the last.
